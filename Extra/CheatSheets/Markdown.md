
Here’s a **clean and handy Markdown cheat sheet** for quick reference:

---

## 📝 Basic Formatting

|Syntax|Output|
|---|---|
|`**bold**`|**bold**|
|`*italic*`|_italic_|
|`~~strikethrough~~`|~~strikethrough~~|
|`` `inline code` ``|`inline code`|
|`[text](url)`|[text](https://example.com/)|

---

## 🔢 Headings

```markdown
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

---

## 📋 Lists

**Unordered List:**

```markdown
- Item 1
- Item 2
  - Subitem
```

**Ordered List:**

```markdown
1. First
2. Second
   3. Subitem
```

---

| Header 1 | Header 2 |

| Cell 1   | Cell 2   |
